Fixing the 'each line' problem

Shell script


IFS=$'\n'
PROCESSES=`ps ux`
for process in $PROCESSES
do
  if [ -n "`echo $process | grep ^$USER`" ]
  then
    echo $process
  fi
done

Current processes from my user

Shell script


IFS=$'\n'
PROCESSES=`ps ux`
for process in $PROCESSES
do
  if [[ $process =~ ^$USER.*$ ]]
  then
    echo $process
  fi
done

Bash-based code

Shell script advantages

  • Easy to call/nest powerful system commands and executables
  • Native support in UNIX systems

Shell script drawbacks

  • No modern language features available
  • Weird syntax

Python-based alternatives

Pure python


#!/usr/bin/env python
from os import listdir
listdir()
for file in listdir():
    print(file)

Simply listing files

Pure python


import os
os.system("ls -la")

Calling the ls command

Pure python


import subprocess
p = subprocess.Popen(['ls', '-la'], stdout=subprocess.PIPE)
(out, err) = p.communicate()
print(out.decode())
